## 5. 진수

- 음의 정수 표현법
  - 2의 보수 표현 : 1의 보수 방법(부호비트를 제외한 나머지 비트를 0, 1 로 표현)으로 표현된 값의 최하위 비트에 1을 더한다.

```python 
# 8진수를 10진수로....
arr=[1,3,5]
result = 0
for val in arr:
    result = result * 8 + val
print(result)

# 문자열을 10진수로...
arr = '135'
result = 0
for val in arr:
    result = result * 10 + ord(val) - ord('0')
print(result)
```
